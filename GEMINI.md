# BestWay プロジェクトの記録

## 概要
AI と人間の間を翻訳する「理解支援OS」。
質問できない初心者が、AI と会話しながら自分のペースで開発スキルを身につけ、ロードマップを可視化するアプリケーション。

## 技術構成
- **フロントエンド**: Next.js (TypeScript) - UI/UX
- **バックエンド**: FastAPI (Python) - AI エージェント / APIサーバー
- **データベース**: Supabase (PostgreSQL)
- **管理・バージョン**: Git / GitHub (Git でのバージョン管理は成功済み)
