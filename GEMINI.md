# BestWay プロジェクトの記録

## 概要
AI と人間の間を翻訳する「理解支援OS」。
「何がわからないか分からない」状態のユーザーに対し、AI が状況を整理・翻訳し、学習ロードマップの可視化と最適な質問補完を行うことで、自走的な学習をサポートするアプリケーション。

## 開発の動機・経緯
- **課題意識**: AI は進化したものの、肝心の人間側が「AI に何をどう質問すればよいか」が分からないというギャップが存在している。
- **目的**: 質問できない初心者のために、AI が状況を整理し、ゴールまでの道筋を動的に生成することで、学習における不安を取り除き、自己成長を可視化すること。
- **重要性**: 本プロジェクトは7月頭に企業発表を控えており、就職活動におけるポートフォリオの核となる予定。技術習得と並行して、プロダクトとしての完成度を高める必要がある。

## 学習・開発スタイル
- **同時並行**: Python をメイン言語として習得しつつ、MVP開発を進める。
- **段階的成長**: 完璧を求めず、まずは動くものを作るアジャイル開発を採用する。
- **将来への投資**: 学生時代に習得した技術がそのままキャリアの武器になるよう、現代のWeb開発標準（Python + TypeScript + FastAPI + Next.js）を採用する。

## 技術スタック
- **フロントエンド**: Next.js (TypeScript) - UI/UX構築
- **バックエンド**: FastAPI (Python) - AIエージェント / APIサーバー
- **データベース**: Supabase (PostgreSQL)
- **管理・管理**: Git / GitHub
